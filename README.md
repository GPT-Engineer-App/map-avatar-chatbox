# map-avatar-chatbox

A map component with an interface that shows user profile avatars on positions within the map. The avatars can be clicked to open a chat box

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/map-avatar-chatbox.git
cd map-avatar-chatbox

```


This will run a dev server with auto reloading and an instant preview.
